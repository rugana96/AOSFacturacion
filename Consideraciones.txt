A tener en cuenta para el diseño de la API:

- Información del taller fuera, prescindible
- No incluir entidades como coche o usuario, usar un ID como "clave foránea"
- No hacer un endpoint para cada busqueda, usar uno para buscar por matricula, id, etc
- Persistencia independiente: No hay una bbdd global, cada servicio tiene su propio sistema de persistencia

Para crear una imagen docker de Swagger editor:

1. Abrir docker
2. Pull de la imagen:
	docker pull swaggerapi/swagger-editor
3. Run contenedor:
	docker run -d -p 80:8080 swaggerapi/swagger-editor
4. Abrir el navegador y escribir localhost o localhost:80

Para parar la imagen:
1. docker ps
2. Copiar id de la imagen (Primer valor)
3. docker stop id_imagen (id_imagen es el valor copiado en el paso 2)
